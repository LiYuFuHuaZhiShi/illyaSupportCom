    [网站属于非盈利性质]
        设计 Weivi
        前端开发 Weivi
        后端开发 Weivi
        产品 Weivi
    
    [文件结构说明]
        Api 后端程序
        Design 设计

            emmx 系统设计稿
        Product 由于全部自我完成设计和开发 所以不考虑绘制原型

        ini 服务器配置文件
